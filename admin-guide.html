üîß Troubleshooting

Problem: Admin login redirects to wrong page

Solution:

  1. Clear browser cache (Ctrl + Shift + Delete)
  2. Open incognito/private window
  3. Run: node create-admin-real-db.js
  4. Restart server: node server.js
  5. Try again!
Problem: No pending organizers showing

Solution:

  1. Check if server is running
  2. Have someone register as "Event Organizer"
  3. Refresh the page
  4. Check browser console (F12) for errors
Problem: Wrong database!

Remember:

  • ‚úÖ Correct: event_management (with underscore)
  • ‚ùå Wrong: eventmanagement (no underscore)
  • Use scripts with -real-db in the name
