üìã What You Need To Do:

Step 1: Create MongoDB Atlas (5 minutes)

  1. Go to: https://www.mongodb.com/cloud/atlas/register
  2. Sign up for FREE (no credit card)
  3. Create a FREE cluster (M0 tier)
  4. Create database user & save password
  5. Whitelist all IPs: 0.0.0.0/0
  6. Get connection string

Step 2: Configure Render (2 minutes)

  1. Go to: Render Dashboard
  2. Click your backend: event-management-system-6lyo
  3. Click "Environment" tab
  4. Add variable: MONGODB_URI = (your MongoDB connection string)
  5. Add variable: FRONTEND_URL = https://event-management-system-0uwx.onrender.com
  6. Click "Save Changes"
  7. Wait 2-3 minutes for redeploy
